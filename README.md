# DoubanRead
Python Spider for Douban Read

用Python抓取[豆瓣读书](https://read.douban.com/columns/category/all?sort=hot&start=0)的图书信息

包括图书封面，作者及类别，并按类别存放在不同文件夹中

采用requests+BeautifulSoup的组合方式进行采集
